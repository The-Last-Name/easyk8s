### 介绍
- easyk8s是一款引导式安装k8s集群个人小应用
- easyk8s并未打算重复造轮子

### 功能
- 支持Centos7
- 支持 HA 高可用部署K8s集群
- 支持 离线 在线 两种安装模式

### 安装
```
# version="0.0.3"
# wget https://github.com/The-Last-Name/easyk8s/archive/v${version}.tar.gz
# tar -xvf easyk8s-${version}.tar.gz
# cd easyk8s-${version}
# chmod +x easyk8s
# ./easyk8s install
```
